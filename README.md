# Proyecto Trámites Gubernamentales

## Descripción
Proyecto para facilitar trámites gubernamentales mediante inteligencia artificial.

## Estructura del proyecto
- `backend/`: Código del servidor
- `bot/`: Código del bot automatizado
- `frontend/`: Interfaz de usuario

## Requisitos
- Python 3.8+
- Docker (opcional pero recomendado)

## Instalación
1. Clonar el repositorio
2. Crear entorno virtual: `python -m venv venv`
3. Activar entorno virtual: `source venv/bin/activate` (Linux/Mac) o `venv\Scripts\activate` (Windows)
4. Instalar dependencias: `pip install -r requirements.txt`

## Ejecución
- Backend: `cd backend && python app.py`
- Bot: `cd bot && python bot.py`
- Frontend: `cd frontend && python server.py`
